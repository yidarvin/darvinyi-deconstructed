# Clarence H. White — source shortlist

All twelve selected photographs are Library of Congress records whose API
metadata marks the digital item `unrestricted: true`. The works were published
between 1897 and 1909, so they are public domain in the United States. Masters
were retrieved from the LOC's openly served TIFF derivatives, converted without
cropping to JPEG for `raw/clarence-h-white/`; *Maine Marshes* uses the
full-resolution public-domain Commons mirror because the LOC master TIFF cannot
be decoded by the local image reader.

## Retrieval verification

The raw directory contains twelve JPEGs, one for each row above. Each opens
successfully and has a long edge from 5,252 to 10,356 pixels (well above the
1,200-pixel sourcing floor). No source miss remains, so no `NEEDED.md` is
required for this pass.

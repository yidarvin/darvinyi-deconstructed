# Shomei Tomatsu — source shortlist

This working set gathers twelve publicly served, web-sized collection renditions from the Art Institute of Chicago. The museum's public collection API identifies each work as a photograph by Shomei Tomatsu, marks each `is_public_domain: false`, and supplies the acquisition credit reproduced below. The JPEGs were retrieved directly from the museum's public IIIF image service, opened successfully, and measure 843px on their long edge (the current 640px minimum).

**Rights basis:** **fair-use** for every selection. Tomatsu's photographs remain copyrighted; the Art Institute records list the artist and collection-credit line but do not supply a named copyright holder. Each assessment below is specific to the credited public display copy: this nonprofit book uses it transformatively for criticism, scholarship, and composition teaching; the complete frame is needed for the stated compositional analysis; the copy is web-sized; and the linked, credited museum record is not replaced by this use or by the image's availability here.

## Research context

The Art Institute's [Tomatsu search results](https://api.artic.edu/api/v1/artworks/search?q=Shomei%20Tomatsu&fields=id,title,date_display,image_id,is_public_domain,artist_display,credit_line&limit=100) establish this as a coherent museum-held cross-section rather than a set of unattributed web copies. It spans Tomatsu's postwar work in Nagasaki, Tokyo, Kawasaki, Iwakuni, and Yokosuka, allowing the next stage to trace how his frames make social pressure tangible through surface, crowding, reflection, isolation, and abrupt changes of scale.

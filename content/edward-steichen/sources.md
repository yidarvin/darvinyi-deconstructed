# Edward Steichen — sources

This source set concentrates on the first, public-domain part of Steichen's career: pictorialist landscapes and portraits (roughly 1901–08), followed by two early modernist studies from 1920–21. It deliberately excludes the later fashion and celebrity photographs that remain associated with the Steichen estate. Every master below was retrieved from a Wikimedia Commons file record that explicitly declares it **Public domain**, **CC0**, or **No restrictions**; the underlying works were made and published before 1929. The Commons file pages identify the contributing institution or source where supplied (including the Library of Congress and Rijksmuseum records).

## Shortlist

### 01 — *The Pond—Moonlight* (negative 1904; print 1906)
- **Why it matters:** Steichen's signature color-gum landscape reduces the pond, bare branches, and reflected moon to a low-contrast field of blue-green atmosphere, with the dark bank as the image's only firm anchor.
- **Source:** [Wikimedia Commons file record](https://commons.wikimedia.org/wiki/File:Edward_Steichen_(American)_-_(Moonlight_-_The_Pond)_-_Google_Art_Project.jpg)
- **Rights:** Public domain, as marked by Commons; pre-1929 photograph/print.
- **File:** `raw/edward-steichen/01-pond-moonlight.jpg` (4044px long edge)

### 02 — *The Flatiron* (1904)
- **Why it matters:** The building rises as a pale triangular wedge through snow and trees; the steep vertical and foreground branches turn a familiar street view into a deliberately atmospheric, asymmetrical composition.
- **Source:** [Wikimedia Commons file record](https://commons.wikimedia.org/wiki/File:Steichen_flatiron.jpg)
- **Rights:** Public domain, as marked by Commons; 1904 work.
- **File:** `raw/edward-steichen/02-flatiron.jpg` (1901px long edge)

### 03 — *Rodin—Le Penseur* (1902)
- **Why it matters:** Rodin and *The Thinker* are paired as two dark, leaning masses against a smoky ground, making sculpture, sitter, and shadow read as one compressed pictorial arrangement.
- **Source:** [Wikimedia Commons file record](https://commons.wikimedia.org/wiki/File:Edward_Steichen_-_Rodin--Le_Penseur_-_Google_Art_Project.jpg)
- **Rights:** Public domain, as marked by Commons; 1902 work.
- **File:** `raw/edward-steichen/03-rodin-le-penseur.jpg` (4193px long edge)

### 04 — *Rodin* (1900–03)
- **Why it matters:** The sculptor's standing figure is isolated in a tall, soft-focus frame, with the pale studio ground and dark coat creating a strong, almost monumental vertical silhouette.
- **Source:** [Wikimedia Commons file record](https://commons.wikimedia.org/wiki/File:Edward_Steichen_-_Rodin_-_Google_Art_Project.jpg)
- **Rights:** Public domain, as marked by Commons; early-1900s work.
- **File:** `raw/edward-steichen/04-rodin.jpg` (4398px long edge)

### 05 — *In Memoriam* (1906)
- **Why it matters:** A kneeling, veiled figure and the opening of a dark architectural threshold arrange grief as a few large tonal planes—an archetypal pictorialist use of silhouette and negative space.
- **Source:** [Wikimedia Commons file record](https://commons.wikimedia.org/wiki/File:Edward_Jean_Steichen_-_In_Memoriam_-_Google_Art_Project.jpg)
- **Rights:** Public domain, as marked by Commons; 1906 work.
- **File:** `raw/edward-steichen/05-in-memoriam.jpg` (4076px long edge)

### 06 — *Road into the Valley—Moonrise* (negative 1904; print 1906)
- **Why it matters:** The road leads in a long, light curve toward the moonlit distance; its single recession line cuts through the otherwise soft, flattened pictorialist landscape.
- **Source:** [Wikimedia Commons file record](https://commons.wikimedia.org/wiki/File:Edward_Steichen_(American)_-_(Road_into_the_Valley_--_Moonrise)_-_Google_Art_Project.jpg)
- **Rights:** Public domain, as marked by Commons; pre-1929 photograph/print.
- **File:** `raw/edward-steichen/06-road-into-valley-moonrise.jpg` (4184px long edge)

### 07 — *On the House Boat—“The Log Cabin”* (1908)
- **Why it matters:** The houseboat's bright cabin, rail, and reflection make a calm stack of horizontal bands, while the mooring lines and tree shadows keep the frame from becoming merely decorative.
- **Source:** [Wikimedia Commons file record](https://commons.wikimedia.org/wiki/File:Edward_Steichen_-_On_the_House_Boat--%22The_Log_Cabin%22_-_Google_Art_Project.jpg)
- **Rights:** Public domain, as marked by Commons; 1908 work.
- **File:** `raw/edward-steichen/07-on-houseboat-log-cabin.jpg` (2557px long edge)

### 08 — *Self-Portrait* (1901)
- **Why it matters:** Steichen turns the photographer into a dark, close-cropped vertical presence; the hand, jacket edge, and shallow illuminated face establish a compact study of pictorialist portrait lighting.
- **Source:** [Wikimedia Commons file record](https://commons.wikimedia.org/wiki/File:Edward_Steichen_(American_-_Self-Portrait_-_Google_Art_Project.jpg)
- **Rights:** Public domain, as marked by Commons; 1901 work.
- **File:** `raw/edward-steichen/08-self-portrait.jpg` (4271px long edge)

### 09 — *Portrait of Eleonora Duse* (1903)
- **Why it matters:** Duse's dark profile, pale head covering, and hand at the lower edge make a spare, sharply directional portrait in which a small highlight carries the emotional center.
- **Source:** [Wikimedia Commons / Library of Congress file record](https://commons.wikimedia.org/wiki/File:Portrait_of_Eleanora_Duse_-_by_Edward_J._Steichen._LCCN2008676241.jpg)
- **Rights:** Public domain, as marked by Commons; 1903 work and Library of Congress record.
- **File:** `raw/edward-steichen/09-eleanora-duse.jpg` (6517px long edge)

### 10 — *Richard Strauss* (1904)
- **Why it matters:** Strauss's head and hand emerge from the studio darkness in an off-center vertical portrait, with the small white cuff and face forming a diagonal of attention.
- **Source:** [Wikimedia Commons file record](https://commons.wikimedia.org/wiki/File:Richard_Strauss_by_Edward_Steichen_1904.jpg)
- **Rights:** Public domain, as marked by Commons; 1904 work.
- **File:** `raw/edward-steichen/10-richard-strauss.jpg` (3300px long edge)

### 11 — *Brancusi's Studio* (1920)
- **Why it matters:** Brancusi's sculptures, plinths, and wall openings are organized as a crowded sequence of verticals and voids—the architectural, object-based clarity that marks Steichen's pivot away from atmospheric pictorialism.
- **Source:** [Wikimedia Commons file record](https://commons.wikimedia.org/wiki/File:Edward_Steichen_-_Brancusi%27s_studio,_1920.jpeg)
- **Rights:** Public domain, as marked by Commons; 1920 work published before 1929.
- **File:** `raw/edward-steichen/11-brancusis-studio.jpg` (3768px long edge)

### 12 — *Isadora Duncan in the Parthenon, Athens* (1921)
- **Why it matters:** Duncan's raised arms and drapery echo the massive column behind her; figure and architecture are made into a single upward, rhythmic vertical composition.
- **Source:** [Wikimedia Commons file record](https://commons.wikimedia.org/wiki/File:Edward_Jean_Steichen_-_Isadora_Duncan_in_the_Parthenon,_Athens_-_Google_Art_Project.jpg)
- **Rights:** Public domain, as marked by Commons; 1921 work published before 1929.
- **File:** `raw/edward-steichen/12-isadora-duncan-parthenon.jpg` (5576px long edge)

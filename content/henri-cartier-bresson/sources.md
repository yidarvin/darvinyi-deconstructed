# Henri Cartier-Bresson — source shortlist

This working set contains twelve Art Institute of Chicago public-display IIIF renditions, retrieved directly from the museum's public IIIF service and checked as opening JPEGs. The display endpoint supplies web copies of roughly 843px on the long edge—above the project's 640px floor but deliberately too modest to function as production-print substitutes.

**Rights basis:** **fair-use** for every selection. The Art Institute API marks each listed work `is_public_domain: false` and identifies Henri Cartier-Bresson as artist. Its records give acquisition-credit lines, not a copyright notice or named rights holder; this book therefore does not attribute an unverified Magnum copyright notice to the records. Each entry below gives a four-factor rationale for the web-sized display rendition used solely for criticism, scholarship, and composition teaching.

## Research context

The Fondation Henri Cartier-Bresson identifies *Place de l'Europe, behind the Saint-Lazare station, Paris, France* (1932) as a signature work and its 2017 *Images à la Sauvette* press material describes the 1952 book as the project whose American title made “The Decisive Moment” inseparable from Cartier-Bresson's reputation: https://www.henricartierbresson.org/en/publications/a6-notebooks/ and https://www.henricartierbresson.org/wp-content/uploads/2016/12/Press-Kit-Images-a%CC%80-la-Sauvette-FHCB-GB.pdf. The present acquired set broadens that single famous jump into a teachable sequence of architecture, delay, crowding, and shifting human scale, while retaining only publicly served, web-sized institutional display copies.

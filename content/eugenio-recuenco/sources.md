# Eugenio Recuenco — source record

## Selection and acquisition note

This is a focused, compositionally varied set from Recuenco's official
*Dreams* project. The portfolio identifies the project but does not publish
individual work titles or making dates, so the stable on-page image identifiers
(`infantil1` etc.) are retained below as *Dreams, Infantil N* and dated
**undated** rather than assigning unsupported titles or dates. The public
portfolio page was checked on 2026-07-24 and each direct JPEG below was
downloaded unchanged from the artist's own public WordPress media endpoint.
All twelve files opened successfully; their long edges range from 1,100 to
1,500px, above the 640px source floor.

The artist's site states: “All images and films on this site belong to Eugenio
Recuenco. All work is protected by copyright.” Accordingly, every selection is
a separate **fair-use** assessment. (1) Each web-sized copy is used for
criticism, scholarship, and transformative composition teaching, not as a
decorative substitute. (2) These are highly creative, copyrighted staged
photographs, a factor weighing against use. (3) The complete frame is necessary
to analyze its staging, cropping, spatial hierarchy, light, and colour; a detail
would defeat that purpose. (4) The chapter credits and links to Recuenco's
portfolio and uses a web-sized reference copy, so it neither replaces the
artist's original nor a licensed reproduction or edition.

**Project/source page:** [*Dreams* — Eugenio Recuenco official portfolio](https://eugeniorecuenco.com/personal_project/suenos-dreams/).  The site-wide copyright notice is visible on the [official *365°* project page](https://eugeniorecuenco.com/trabajos/365/).

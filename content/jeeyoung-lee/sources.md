# JeeYoung Lee — source shortlist

JeeYoung Lee’s *Stage of Mind* pictures are completed, room-sized hand-built installations photographed as intact scenes, then dismantled. This source set retains ten public, artist-attributed web display images from LensCulture’s *Stage of Mind* presentation. The presentation supplies the title, date, medium, copyright notice, and whole-image public URL for every selection; each downloaded JPEG opens successfully and has a 1000px long edge. No open-license or public-domain reproduction was located for these copyrighted works.

**Rights basis:** **fair-use** for every selection. Each is an individual, limited assessment: this nonprofit book uses a credited, linked, web-sized copy for transformative criticism, scholarship, and composition teaching; the complete published frame is necessary to analyze Lee’s constructed spatial relations; and the derivative does not substitute for the artist’s work, a gallery record, or a licensed reproduction. LensCulture’s project page, which identifies every image as © JeeYoung Lee, remains linked below.

## Research context

LensCulture’s artist-published presentation says Lee handcrafts every element of these psychological landscapes, puts a model into the finished set, photographs it, and then demolishes the scene: https://www.lensculture.com/projects/1874039-stage-of-mind. Echo Fine Arts’ current artist page also documents the *Stage of Mind* series: https://echofinearts.com/artists/25-jeeyoung-lee/series/stage-of-mind/. The gallery’s listing of “Blackbirds” as 2011 conflicts with LensCulture’s 2009 caption for the supplied *Black Birds* image, so this chapter attributes its work dates to the linked LensCulture presentation rather than treating the gallery page as confirmation of every selected date.
